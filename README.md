# Palaeohispanic Dataset Generator

This project contains the scripts that transform Hesperia's Database data into a ML-ready dataset.

The input data has been extracted from multiple sources, with their respective licenses:

- [`ES_cities.csv`](https://centrodedescargas.cnig.es/CentroDescargas/detalleArchivo?sec=9000004): NGMEP CC-BY 4.0 ign.es
- [`FR_cities.csv`](https://www.data.gouv.fr/datasets/villes-de-france/informations): It is made available here under the Open Database License (ODbL).
- `celtiberian_raw.csv` and `iberian_raw.csv` were extracted from the [Hesperia Databank](http://hesperia.ucm.es/presentacion.php) on October 2025
- `provinces.csv` is a document compiled from the information available in Wikipedia about the centroids of the Spanish Provinces and French Departments.

## Scripts

### [corpus_preprocess.ipynb](./corpus_preprocess.ipynb)

This is a Python Notebook that describes the processes needed to convert Hesperia's data into machine-readable values.

### [dual_system_dataset_preprocess.ipynb](./dual_system_dataset_preprocess.ipynb)

This file is also a Python Notebook in which the base corpus generated by `corpus_preprocess.ipynb` is transformed into a useful dataset for the exploration of experiments regarding the dual system of the Palaeohispanic inscriptions.
